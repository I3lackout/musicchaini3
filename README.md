# MusicChain i3

## Features

| | |
|---|---|
| 🎵 **Music streaming** | Tracks, library, playlists, native mobile app (beta) |
| 🎥 **Live sessions** | Host and join live streams with WebRTC and watch rewards |
| 💎 **Resource economy** | McCC · Fuel · Gas · ITC — tier-adjusted, documented rates |
| 🔗 **Track blockchain** | Auditable blocks, triad commits, MusicChain logs |
| 📊 **Analytics** | Listening stats, TFPS field, PIA/TIA interaction analytics |
| 🤝 **Community driven** | Matrix ranking, scoreboard, seed-a-node, MCIP standards |

## Installation

This repository is **documentation-first** (Phase 1). There is no single `git clone && run` for the full production stack yet.

| What | Where |
|------|--------|
| **Docs & specs** | Clone this repo — start at [docs/README.md](./docs/README.md) |
| **Tokenomics reference** | [musicchain-tokenomics/](./musicchain-tokenomics/) |
| **API reference (stub)** | [musicchain-api/](./musicchain-api/) |
| **Mobile app (beta)** | Install via [musicchaini3.com/app](https://musicchaini3.com/app) |
| **Full backend** | Private monorepo — not published in Phase 1 |

Public SDKs and extractable packages are on the [roadmap](./ROADMAP.md) (Phase 2–4).
