# MusicChain i3

**Stream. Listen. Earn. Verify.**

MusicChain i3 is a music streaming platform where listening, creating, and participating in the ecosystem generate **measurable resources** — McCC, Fuel, Gas, and ITC — recorded on an auditable **track blockchain** and in **MusicChain logs**.

This repository is the **public face** of the project: documentation, standards, and (over time) extractable open-source modules. The production Flask application remains private during early phases.

- **Live site:** [musicchaini3.com](https://musicchaini3.com)
- **Ecosystem index:** [ECOSYSTEM.md](./ECOSYSTEM.md)
- **Standards (MCIPs):** [musicchain-specifications/](./musicchain-specifications/)
- **Tokenomics reference:** [musicchain-tokenomics/](./musicchain-tokenomics/)

## What is MusicChain i3?

A **digital music ecosystem** with:

| Layer | Description |
|-------|-------------|
| **Streaming** | Tracks, library, playlists, mobile app |
| **Live** | Live sessions, watch rewards, WebRTC |
| **Economy** | McCC · Fuel · Gas · ITC with tier-adjusted rates |
| **Chain** | Track blocks, sub-blocks, MusicChain logs |
| **Identity** | Track soul, triad commits, TFPS field analytics |
| **Community** | Matrix ranking, scoreboard, seed-a-node, notifications |

We optimize for **transparency**: rules are documented here so anyone can understand *what the system claims* and *how rewards are supposed to work*.
